All ABOUT AI CHATBOTS
OVERVIEW

This project is a Java Swing application designed to provide information about AI chatbots in an interactive way. The app includes a main page with educational content, an interactive animation, and a quiz feature with 10 questions.

FEATURES

Main page that introduces AI chatbots, their purpose, and how they work

An interactive animation that users can click to explore engagement features

A quiz section with 10 multiple-choice questions to test knowledge, including scoring and feedback

TECHNOLOGIES USED

Java (JDK 8+)

Java Swing for GUI components
